# Themes

gdluxx includes multiple color themes. Pick whichever matches your style and
mood.

## Changing Your Theme

### Step 1: Go to Settings

Click **Settings** in the main menu, then click **General Manager**.

### Step 2: Find the Theme Selector

In **"Theme Selection"** you'll see theme options displayed as cards.

### Step 3: Click Your Preference

Click the theme card you like. Changes apply immediately.

## Available Themes

gdluxx ships with seven built-in themes. Each one supports both light and dark
mode (see [Modes](#modes) below).

- **Indigo** — Enhanced indigo _(default)_
- **Media Gallery** — Orange-focused
- **Developer Tool** — GitHub-inspired
- **High Contrast** — Maximum contrast
- **Media Downloader** — Amber-focused
- **Terminal Dark** — Green-focused
- **Power User** — Purple-focused

## Modes

Every theme has a dark and light mode. You'll see the toggle for it in the upper
right hand area of the browser window.
